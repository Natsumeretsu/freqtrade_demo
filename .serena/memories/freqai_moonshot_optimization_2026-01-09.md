# FreqAI Moonshot 策略优化记录

> 日期：2026-01-09
> 策略：FreqaiMoonshotStandalone
> 配置：configs/freqai/lgbm_moonshot_futures_v1.json

## 优化背景

原版策略表现：
- 总收益：+44.34%
- 胜率：71.8%
- **Profit Factor：1.12**（过低，目标 >1.5）
- Win/Loss Ratio：0.45（平均赢 6.88%，平均亏 15.40%）
- 止损单贡献 -369.75 USDT，吃掉大部分利润

核心问题：**"吃如雀，拉如象"** - 高胜率但盈亏比差。

---

## 优化尝试记录

### 1. 止损优化（全部失败）

| 方案 | 收益 | 胜率 | 结论 |
|------|------|------|------|
| 原版 (15%止损) | +44.34% | 71.8% | 基准 |
| 方案A (trailing stop) | +26.30% | 79.0% | 更差 |
| 方案B (阶梯式custom_stoploss) | -83.32% | 33.5% | 爆仓 |

**结论**：止损优化行不通，策略需要"呼吸空间"。

### 2. 入场优化（成功）

**三刀优化**：
1. 剔除 ADA/DOT（贡献 77% 止损单）
2. BTC 趋势过滤器（已存在）
3. 提高预测阈值 0.6% → 0.8%

| 配置 | 收益 | 胜率 | 回撤 |
|------|------|------|------|
| 原版 4币 | +44.34% | 71.8% | 19.64% |
| 0.8% + 2币 | +8.03% | 74.1% | 6.77% |
| **0.8% + 2币 + 3x** | **+9.34%** | **84.6%** | **3.60%** |

### 3. 杠杆测试

| 杠杆 | 收益 | 胜率 | 回撤 | 结论 |
|------|------|------|------|------|
| 3x | +9.34% | 84.6% | 3.60% | 最优 |
| 5x | +8.03% | 74.1% | 6.77% | 可接受 |
| 10x | -13.4% | 60.9% | 19.49% | 爆仓风险 |

## 最终固化配置

### 策略参数 (FreqaiMoonshotStandalone.py)

```python
# 止损配置
stoploss = -0.15  # 固定 15% 止损
trailing_stop = False
minimal_roi = {"0": 0.10}  # 10% 止盈

# 入场参数
buy_pred_threshold = 0.008  # 0.8% 阈值
buy_max_atr_pct = 0.10      # 收紧波动率

# 杠杆配置
leverage_value = 3.0  # 3倍杠杆
```

### 配置文件 (lgbm_moonshot_futures_v1.json)

```json
"pair_whitelist": [
  "BTC/USDT:USDT",
  "ETH/USDT:USDT"
]
```

## 关键洞察

1. **止损不能收紧**：策略预测逻辑需要"深呼吸"空间
2. **入场质量 > 出场优化**：过滤垃圾交易比优化止损更有效
3. **杠杆上限 3x**：配合 -15% 止损，强平线在 -33%，安全
4. **ADA/DOT 不适合此模型**：波动特性与模型不匹配
